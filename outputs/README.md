# Outputs

This directory contains regenerated Python reproduction artifacts. The
authoritative inputs remain the tracked paper, R scripts, R data files, and R
reference outputs under `cross-section/` and `panel/`.

Regenerate and validate the complete artifact set with:

```bash
python scripts/run_all.py
python scripts/validate_project.py --generated-outputs
```
