# AI Disclosure

AI assistance was used to support coding, refactoring, debugging, and
documentation for this reproduction project. The AI-assisted work included
project scaffolding, path validation, Python ports of the R workflows,
Docker/reproducibility wiring, test suggestions, and reviewer-facing Markdown
drafts.

Models used: OpenAI Codex, GPT-5 class coding assistant, accessed through the
local development environment; and Anthropic Claude Code, Claude Opus 4.7
(1M context), accessed through the Claude Code CLI.

Human contributors reviewed the generated code and documentation, compared the
Python outputs against the thesis, original R workflows, and exported R tables,
and decided which results and caveats belong in the final submission. The final
interpretation of the empirical results remains the responsibility of the
project team.

No externally sourced code snippets were copied verbatim into the project. The
source and formula annotations for statistical methods, model APIs, and
package-dependent approximations are recorded in `docs/source_annotations.md`.
